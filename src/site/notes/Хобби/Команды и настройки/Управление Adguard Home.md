---
Для запуска Adguard Home на роутере необходимо подключиться по ssh к Entware порт 222 и использовать команду:

```console
/opt/etc/init.d/S99adguardhome start
```

Для остановки 

```console
/opt/etc/init.d/S99adguardhome stop
```
