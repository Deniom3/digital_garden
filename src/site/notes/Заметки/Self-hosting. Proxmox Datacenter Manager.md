Единый интерфейс управления и мониторинга для [[Заметки/Self-hosting. Proxmox Virtual Environment\|PVE]] и [[Заметки/Self-hosting. Proxmox Backup Server (PBS)\|pbs]] с возможностью мониторинга ресурсов и задач, а так же управления контейнерами и виртуальными машинами в ограниченном объеме.

Позволяет удобно в едином интерфейсе управлять состояниями виртуальных машин, отслеживать обновления и нагрузку на ноды proxmox без объединения в кластер.

> [!important]
> Главной фишкой является возможность легкой миграции между нодами добавленными в PDM даже если они не объединены в один кластер.

Документация: https://pdm.proxmox.com/docs/introduction.html
